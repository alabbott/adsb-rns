# adsb-rns

ADS-B sender / receiver for the [Reticulum Network Stack](https://reticulum.network/). 

I originally started this project as a vibe-coded endeavor which can be found here [alabbott/adsb-radar](https://github.com/alabbott/adsb-radar).

I have decided to take a more intentional approach to developing this project. Some parts of the original will be reused. Where AI code is used I will make notes. All code will be hand written and reviewed to the best of my knowledge. If it tanks the network at least there will be a human to blame :)
